## Install and use

- Get the project code

#### github

```bash
git clone https://github.com/fbljq5/mix-vue.git
```

#### gitee

```bash
git clone https://gitee.com/RoyLiang/mix-vue.git
```

- Installation dependencies

```bash
cd mix-vue

yarn install

```

- run

```bash
yarn serve
```

- build

```bash
yarn build
```
