## 安装使用

- 获取项目代码

#### github

```bash
git clone https://github.com/fbljq5/mix-vue.git
```

#### gitee

```bash
git clone https://gitee.com/RoyLiang/mix-vue.git
```

- 安装依赖

```bash
cd mix-vue

yarn install

```

- 运行

```bash
yarn serve
```

- 打包

```bash
yarn build
```
